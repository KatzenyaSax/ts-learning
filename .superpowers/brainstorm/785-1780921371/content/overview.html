三路分区快速选择 (QuickSelect)

荷兰国旗问题 + 剪枝 = 平均 O(n) 的第 k 大查找

一句话总结

每次选一个 pivot，把数组切成三块：
⬅ 比 pivot 小的  |  🟦 等于 pivot 的  |  比 pivot 大的 ➡

然后看目标落在哪一块，只递归那一块（或直接命中返回）。

为什么需要三路分区？

二路分区（Lomuto）的问题
当数组全是 1 时：
arr[i] < pivot 永远为 false

→ 所有元素在 "≥ pivot" 一侧
→ 范围每次只缩小 1
→ n 层递归 → O(n²) 超时
三路分区的解决方案
当数组全是 1 时：
所有元素归入 "== pivot" 区域

→ targetIdx 一定落在 ==pivot 中
→ 一次扫描后直接返回
→ O(n)，0 次递归
